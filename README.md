# Python brainfuck compiler
## !!! Linux only !!!

`USAGE: pbfc.py [path_to_source]`

pbfc.py should generate `a.out` executable as a result. Don't call important files as `a.out` please.

Dependencies:
 - Python 3
 - nasm
 - ld
